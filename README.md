# PalindromeApp
📌 Overview

Palindrome Checker App is a Java-based project that demonstrates multiple approaches to check whether a string is a palindrome.

Each approach is implemented as a separate Use Case (UC1–UC13) and managed using a proper Git branching workflow.

⸻

🚀 Use Cases Implemented
	•	UC1: Application Entry & Welcome Message
	•	UC2: Hardcoded Palindrome
	•	UC3: String Reverse Method
	•	UC4: Character Array Approach
	•	UC5: Stack-Based Implementation
	•	UC6: Queue + Stack Approach
	•	UC7: Deque-Based Approach
	•	UC8: Linked List Approach
	•	UC9: Recursive Method
	•	UC10: Case-Insensitive & Space-Ignored Check
	•	UC11: Object-Oriented Design
	•	UC12: Strategy Design Pattern
	•	UC13: Performance Comparison
